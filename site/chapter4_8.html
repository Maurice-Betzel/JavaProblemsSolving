I added FILENAME paremeter to be able to run with different files. This is how to read parameters in the applet class. 

                    String paramFile = getParameter("FILENAME");
                    if (!paramFile.isEmpty())
                        numberSourceFile = paramFile;

Another change is to catch possible exceptions.

  • Cached IOException when trying to open the file. 
                            try{
                            .

                        }
                        catch(IOException e){
                            txtResult.append("\nCould not read file " + numberSourceFile);
                        }
  • Cached exception when trying to convert a line in the file to string. 
                            try{
                            .

                        catch (NumberFormatException nfe)
