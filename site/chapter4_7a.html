Move each of the Java applications in Exercises 4.1-4.3 to a package named myprog.single. Compile and run each of the applications.


We need to compile applet class considering package definition. 

                    C:/> javac myprog\single\Factorial.java

To run the applet, we need to specify applet class including package names. 

                code="myprog.single.Factorial.class"

To run the application, we need to specify applet class when running java. 

                java myprog.single.StudentEmail students.txt studentemail.txt
